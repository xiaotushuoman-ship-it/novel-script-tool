# 抖音爆款短剧提示词模板

## 设计原则

- **动态学习原则**：用户提供的参考视频、截图、提示词、分镜，只学习其中能提升成片质量的优点，摒弃会造成AI感、模板感、过度复杂、站位混乱、动作迟钝的缺点。禁止机械照抄参考格式。
- **先判断再采用**：每次吸收参考内容，先判断它解决的是画质、节奏、站位、动作、转场、声音、角色一致性还是情绪表达问题；只把适合当前题材和生成模型能力的部分加入输出。
- **反模板化原则**：不得把任何一种分镜结构固定套到所有题材。秒表式战斗、三段式广告、纪录片空境、单镜头一镜到底、快剪蒙太奇都只是可选工具，必须按剧情密度和生成稳定性动态选择。
- **优点拆解原则**：参考内容必须拆成可迁移方法，例如“色彩总控”“站位锚点”“动作结果”“声音匹配剪切”“单镜头主体明确”，而不是复刻人物、场景、品牌、水印、字幕或软件界面。
- **缺点过滤原则**：参考内容里如果存在字数过长、镜头过多、动作过密、暴力过度、光效堆砌、版权/品牌复刻风险、不可生成的复杂动作，必须主动压缩、替换或删除。
- **自我进化原则**：用户指出问题后，必须更新后续输出策略；同类错误不得反复出现，例如固定尘土转黑、漏写色彩影调、只列问题不直接给可用修正版。
- 前3秒只做一件事：给冲突、反差、危险、身份错位、情绪爆点。
- 每组小于等于15秒，只拍一个事件，不把整段小说塞进一个镜头。
- 每3-5秒换一个信息点：动作、反应、误会、反转、悬念。
- 每个镜头单独可生成：人物、场景、动作、表情、光线、运镜必须明确。
- 固定角色资产：每次重复角色外貌、服装、身份锚点，避免变脸。
- 场景只选一个：一个视频提示词内不混多个空间。
- 先锁定空间站位，再写镜头：人物可以被镜头从不同角度拍，但位置不能漂移。
- 前后分镜必须继承站位锚点：当前组开头承接上一组结尾，当前组结尾留下下一组可接的动作和位置。
- 人物互动必须准确：谁看谁、谁跪谁、谁逼近谁、谁回避谁、谁等待谁回应，都要写清楚。
- 微表情保持克制：优先用视线、唇线、下颌、呼吸、手指、重心变化，少用冷汗和眼泪。
- 声音服务剪辑：台词、环境声、动作声、BGM卡点要能对齐。
- 结尾必须留钩子：表情、关键物、门开、来人、危险逼近、身份将暴露。
- 去AI感优先级高于炫技：人物像真人表演，镜头像真实摄影，光线来自场景内可解释光源，材质有重量和瑕疵。
- 即梦长度限制：每组分镜提示词总字数不超过1500字，优先保留站位、互动、动作链、光线和材质，删除解释性文字。
- 角色称呼必须使用角色全名或明确角色名；禁止用男人、女人、男主、女主、他、她、它等泛称和代词。
- 角色称呼规则只约束分镜描述和提示词；台词按真人说话逻辑，可以自然使用“我、你、他、她”等代词。
- 负面提示词默认只写“不要随机换脸，不要字幕水印”；除非用户内容涉及明显审核风险或技术问题，不额外堆限制。
- 生成正式提示词前，若用户未指定风格，先提供3个适合剧情的风格选项供用户选择，不要直接固定风格。
- 出现快剪、回忆、闪回、时间跳跃、意识片段时，可适当使用剪映高级转场效果；转场必须服务叙事，不要滥用。
- 分段硬规则：用户要求生成提示词时，每一段内容必须小于等于15秒；剧情超过15秒必须自动拆成新段。每段开头必须参考上一段结尾站位锚点，重新确认人物位置、朝向、距离和互动关系。

## 镜头写法公式

`谁 + 在哪里 + 做什么 + 脸/手/身体的微反应 + 环境反馈 + 镜头为什么移动 + 落点`

例：
`林凡盯着石门缝隙，瞳孔缩紧，右手攥住法袍衣襟，指节发白，门框震落石粉，镜头贴着尘土向门缝推进，落在扩大的金光上。`

## 连续站位规则

- 先定义固定坐标：以前方/后方/左侧/右侧/高处/低处/门槛/桌面/走廊等空间参照物锁定人物。
- 当前组第一镜必须承接上一组最后一镜的人物位置、朝向、动作余势和视线方向。
- 当前组内，除非当前文案明确写移动，人物不得突然换边、换距离、换高低关系。
- 镜头可以从正面、侧面、低角度、俯角拍摄，但必须说明这是换机位，不是人物换位。
- 当前组最后一镜必须留下下一组能接住的站位锚点。
- 第二段及以后，必须先写“承接上一段结尾站位”：上一段最后谁在哪、朝向谁、距离多少、手/脚/武器/道具处于什么状态。
- 新段人物位置可以随剧情变化，但必须从上一段结尾的动作余势自然变化，不能跳跃换位。

## 人物互动规则

- 每个互动句都写清对象：A看向B，A跪向B，A挡住B，A等待B回应，A回避B视线。
- 对话时，说话者和听话者都要出现：说话者有嘴部动作，听话者有反应特写。
- 压迫关系要可见：谁站着谁跪着，谁在高处谁在低处，谁逼近谁后退，谁占画面中心谁被压边。
- 误会流场景要写双层关系：表面互动是什么，真实心理反差通过可见动作表现。
- 角色称呼只用角色全名或明确角色名，例如“林凡”“迪长老”“苏清月”；不要写“男人、女人、男主、女主、他、她、对方”。
- 台词按角色真实口吻写，可以出现“我、你、他、她”等代词，不要为了角色名规则破坏台词自然度。

## 微表情规则

- 优先使用：视线停顿、眼神避开半秒、唇线变紧、下颌僵住、呼吸停住、手指捏紧衣料、重心轻微后移。
- 少用冷汗；只在生死危机或身份即将彻底暴露时使用，且一组最多一次。
- 不用眼泪，除非原文明确悲伤、诀别、痛哭。
- 不写夸张脸谱化表情，如“吓到变形”“满头大汗”“泪流满面”。

## 去AI感规则

### 表演真实

- 人物反应要有顺序：先听到/看到，再停顿，再微反应，再动作。
- 情绪不直接写满，要控制幅度：轻微皱眉、短暂停顿、嘴唇收紧、眼神避开、手指停住。
- 避免所有角色同时做夸张动作。群像只让前景1-2个人有明确反应，背景保持自然延迟。
- 台词之后必须有短反应，不要让角色说完立刻机械切镜。

### 摄影真实

- 镜头移动必须有目的：为了看清眼神、手、门缝、法袍、白骨、跪拜关系才移动。
- 不要每镜都推拉摇移。静态压迫镜头、轻微手持、短距离推近更像真实拍摄。
- 快速运动只用于动作峰值；情绪镜头用慢推或固定镜头。
- 镜头落点必须清晰：眼睛、手指、门缝、关键物、人物站位关系。

### 光线真实

- 每组只保留1-2个主光源：烛火、门缝逆光、月光、火光、窗光等。
- 光线必须跟空间一致：门外光只能从门缝方向进入，烛火只能在近处产生暖光跳动。
- 少用无法解释的全身发光、彩色粒子、凭空光柱。
- 暗部保留细节，不写“全黑”“全白爆光”。

### 材质真实

- 写材质重量：法袍厚重、衣摆拖地、石粉落下、木门震动、金属轻响。
- 写瑕疵：灰尘、褶皱、磨损边缘、旧纹路、细小划痕，比“完美高级”更真实。
- 避免塑料感词汇：炫酷铠甲、华丽光效、梦幻粒子、超级能量。

### 剪辑真实

- 每组只保留一个明确爆点，不要同时塞身份揭露、战斗、系统、回忆、群像反应。
- 切点落在动作结果：抬眼后、手指收紧后、石门震动后、跪拜落地后。
- 最后一镜的钩子要来自原文事件，不要额外编危险。

## 动作战斗参考规则

用于 AI动漫战斗短片、武侠对决、玄幻打斗、追逐、闪避、拔剑、爆发类镜头。

- 动作不能只写结果，必须写动作链：起势、发力、接触、反馈、收势。
- 每个打斗镜头只拍一个动作峰值：拔剑、踏步、闪避、格挡、撞击、落地、回头。
- 先锁定双方站位：A在左前，B在右后，距离几步，谁主动进攻，谁防守后撤。
- 攻防关系必须清楚：A挥向B，B抬臂格挡；A冲向B，B侧身避开；A落地，B被气流逼退。
- 镜头跟随动作原因：为了看清脚步发力、手腕转向、武器接触点、落地反震才移动。
- 动作反馈必须物理可见：衣袖被拉直、碎石弹起、尘土向外散、发丝滞后、脚掌压出灰痕。
- 速度控制：近身格斗默认实时速度，不写慢动作、定格、停顿、急停、轻震。用“连续出招、马上接下一拍、动作峰值切镜”保持速度。
- 不要无意义乱晃。动作镜头使用“稳底盘 + 短促跟拍”，落点必须清楚。
- 多人战斗不要同时乱动。前景主动作清楚，背景人物做延迟反应或保持防备姿态。
- 少用满屏爆炸和全身光效；优先用身体力学、武器碰撞、布料、尘土、环境破坏制造爽感。

动作镜头公式：

`站位关系 + 起势动作 + 发力部位 + 接触/闪避点 + 物理反馈 + 立即接下一动作`

例：
`黑衣少年站在画面左前方，右脚向后压住碎石，肩线下沉，腰胯带动手臂斜向挥剑；剑刃擦过对手抬起的护臂，火星短促弹开，衣袖被冲击拉直，对手借格挡力侧身换位，镜头跟着侧身动作切到下一招。`

## 国漫高密度打斗语法

借鉴顶级国漫动作片的通用打斗语言，不复刻具体IP角色、场景、招式或标志性镜头。

- 节奏：一秒内可以有2-3个动作信息，但只保留一个清晰主体动作。
- 站位：先给双方轴线，随后用绕身、错肩、贴地滑步、翻身落位改变攻防角度，不让人物随机换边。
- 连续性：防守不是结束，而是下一招的起点；格挡后借力转身，闪避后顺势贴近，落地后马上补步。
- 顿挫：动作不是慢，而是“快-更快-接触-立即变向”；不要写停顿，写“接触后借力改向”。
- 身体发力：脚掌抓地、膝盖压低、腰胯拧转、肩线甩出、手腕翻转，按身体链条写。
- 水墨/气流拖尾：只作为动作轨迹辅助，像衣袖残影、尘线、气流弧线，不能盖住人物动作。
- 地面反馈：碎石、尘圈、水痕、草叶倒伏、衣摆反向甩动，用来证明动作速度和力量。
- 镜头：近身拆招用手臂和肩线近景，爆发位移用低机位中景，落地反震用脚步特写。
- 镜头切点：切在“接触后变向”的瞬间，不切在摆姿势瞬间。
- 强弱关系：高手不大喊乱挥，动作短、准、贴身；弱者动作大、重心乱、需要补步。
- 运镜：空间关系稳定不等于固定站桩。允许跟随动作、过肩、主观POV、第一人称、FPV穿越机式掠过、贴地滑行、绕身半圈，但每次运镜必须有动作动机和清晰落点。

高密度动作公式：

`轴线站位 + 连续三拍 + 接触借力变向 + 脚步补位 + 气流/尘线显示轨迹 + 下一招已经启动`

例：
`黑衣男在右侧压低重心，橙衣男从左侧直拳抢中线；黑衣男左前臂贴着拳线向外刮开，右脚同时踩进橙衣男外侧半步，肩线擦过橙衣男胸前，右肘顺着转胯短促顶出；橙衣男被迫拧腰卸力，脚跟擦出半圈水痕，黑衣男顶肘后不收势，手腕已经翻向橙衣男后颈外侧准备下一拍。`

## 动作卖点设计

如果用户反馈“像过家家、缺少吸引眼球”，说明动作只有拆招，没有卖点。每个15秒打斗段至少设计一个高辨识动作卖点。

- 卖点动作必须服务剧情胜负，不是炫技插入。
- 卖点动作前要有铺垫：脚步位置、重心方向、环境可借力点。
- 卖点动作后要有结果：对手失衡、被迫换位、撞到环境、武器脱手、距离被压缩。
- 卖点动作不能脱离空间：墙、桌、竹竿、护栏、石柱、车门、台阶、泥地、水面都可以成为借力点。

常用动作卖点：

- 扫堂腿：低位旋身，脚背贴地扫向对手支撑腿，地面水痕/尘圈显示轨迹。
- 前空翻越位：角色借对手肩膀、墙面、石柱或栏杆发力，从正面越到侧后方。
- 后空翻脱困：角色被逼近时后仰蹬地翻出压迫线，落地马上接反击。
- 墙面借力蹬踢：脚踩墙面或石柱改变方向，身体横向弹出。
- 滑铲穿裆/低位穿身：贴地穿过对手攻击线，从身后起身反扣。
- 肘膝连击：贴身状态下短肘、顶膝连续发生，适合狭窄空间。
- 借物换位：抓住竹竿、栏杆、衣领、枪带、腰带或袖口改变角度。
- 环境压制：撞竹、撞墙、撞护栏、踩碎木板、踢翻椅子，环境反馈必须明确。

动作卖点公式：

`铺垫站位 + 借力点 + 卖点动作 + 轨迹反馈 + 对手结果 + 下一招承接`

例：
`苏清月被沈砚逼到湿竹旁，苏清月左手抓住竹竿借力，右脚贴着泥地扫向沈砚支撑腿；泥水被脚背扫出半圆，沈砚膝盖被迫弯下，苏清月借竹竿反向旋身绕到沈砚侧后方，落地时手已经扣向沈砚肩线。`

### 动作卖点禁忌

- 不写凭空空翻。空翻必须有蹬地、借肩、借墙、借栏杆等发力来源。
- 不写连续多个大招。一个15秒段只保留1个主卖点，其他动作用来铺垫和收束。
- 不写没有结果的炫技。每个翻身、扫腿、滑铲都必须改变双方站位或胜负关系。
- 不让镜头只看脸。卖点动作必须看见脚步、腰胯、手臂和环境反馈。

## 高级运镜规则

- 跟随镜头：跟随拳、脚、肩线、手腕、衣袖或角色重心移动，适合连续拆招。
- 过肩镜头：从防守者肩后看进攻者，适合表现压迫和来拳方向。
- 第一人称/POV：只用于短促瞬间，如看到拳头逼近、手腕被扣、脚步卡位；不要整段都主观。
- FPV穿越机式：快速从角色肩侧、手臂缝隙、腿边或门缝掠过，适合位移、闪避、冲刺、穿过人群。
- 贴地掠过：从脚步、水痕、尘土、碎石高度跟随，适合展示发力和速度。
- 绕身半圈：围绕双方身体转半圈，适合贴身缠斗和换位，但必须保持左右关系可读。
- 运镜必须回答：镜头为什么动、跟着谁动、从哪到哪、落到哪个动作点。
- 高级运镜不是乱飞。每组最多使用1-2个复杂运镜，其余镜头保持清楚。

高级运镜公式：

`运镜类型 + 跟随对象 + 路径 + 动作动机 + 落点`

例：
`过肩跟随镜头从黑衣男右肩后切入，沿橙衣男打来的右拳向前压近，拳头被黑衣男左前臂刮开后，镜头顺着前臂滑到两人脚下卡位，落在黑衣男右脚踩进橙衣男外侧的瞬间。`

例：
`FPV穿越机式镜头贴着地面从两人脚边掠过，穿过被鞋底擦开的水痕，沿黑衣男斜跨的脚步绕到橙衣男背侧，落在橙衣男重心被带偏的肩线。`

## 压制式打斗收束

用于一方逐渐占上风、另一方被逼退/撞上环境/倒地/失去继续战斗能力的片段。参考用户提供的竹林打斗分镜结构。

- 胜负递进：发现破绽 → 贴身撞开重心 → 撞上环境 → 不给喘息继续逼近 → 绊倒压肩 → 倒地挣扎 → 手部失力 → 环境安静。
- 压制关系必须连续：优势方一直向前，劣势方一直被迫后退、侧移、撞上环境或倒地，不要中途重置站位。
- 环境必须参与动作：湿竹震动、水滴落肩、泥水溅起、枯叶被抓皱、雾气回流，用环境反馈代替夸张特效。
- 微表情克制：瞳孔收缩、咬紧牙关、下颌绷住、手指抓泥、呼吸被压住；不用频繁冷汗和眼泪。
- 节奏：前半段连续压迫，后半段逐渐安静，让“环境声回归”体现战斗结束。
- 胜者收手：最后不继续追打，只低头看着对方，胸口轻微起伏，形成冷硬压迫。
- 败者失力：不要写夸张昏迷；用手指松开、枯叶滑落、肩膀贴地、呼吸压低表现结束。

压制收束公式：

`破绽反应 + 贴身撞重心 + 环境撞击反馈 + 追击压肩 + 绊脚倒地 + 手部失力 + 胜者沉默 + 环境归静`

压制收束例：

`女人眼神一紧发现男人压低身体避开竹枝，男人贴近她身侧，用肩膀撞开她的重心，同时手臂压住她持竹枝的手；女人后背撞上湿竹，竹叶水滴震落到肩头。男人不给喘息，跟步逼近，侧身避开反击后扣住肩膀，扫腿绊住脚踝并顺势压低肩线。女人双手撑进泥地，手指抓住枯叶又慢慢松开，男人站在她身后半步处，竹林环境声重新回到画面。`

## 参考视频风格条目：废墟动漫战斗

### 画面风格

- 场景：废墟城市、断裂高楼、低云烟尘、地面积水反光，远景交代空间尺度。
- 色彩：低饱和冷灰为底，角色能量用单一强调色区分阵营，如紫色与红色。
- 光线：强逆光和侧逆光勾人物轮廓，脸部保留暗面，不要全脸均匀打亮。
- 材质：金属甲片有磨损反光，布料有撕裂边缘，地面有湿反射和碎石，不写塑料般完美表面。

### 构图规律

- 开场可用近景压脸：角色占画面大部分，背景废墟虚化，先建立压迫感。
- 战斗前必须给远景站位：双方一左一右，距离、朝向、攻防关系清楚。
- 冲撞瞬间可用强光遮切，但强光只作为动作峰值，不要持续遮住主体。
- 烟尘不是装饰，是空间层：前景烟遮挡、主体清晰、背景废墟虚化。
- 胜负关系用高低差和压脸构图：胜者俯视，败者低位或倒在前景。

### 动作节奏

- 对峙：近景眼神或手部能量蓄势，人物少动，能量和发丝轻动。
- 发起：脚步压地、身体前倾、肩线下沉，先给发力准备。
- 冲撞：快速位移加短暂强光，接触点清楚。
- 反馈：烟尘扩散、地面水纹震开、衣料和头发滞后。
- 收束：用近景停在胜者脸或败者被压制的姿态，减少多余动作。

### 去AI感注意

- 不要让两个角色在同一镜头中同时做复杂动作；一个主动作，一个反应动作。
- 不要让能量球、闪电、烟尘同时抢主体；每镜只选一个视觉重点。
- 不要连续使用爆白镜头；爆白后必须回到清晰站位或脸部反应。
- 不要写“震撼大战”，要写“左侧角色冲向右侧角色，右侧角色后撤半步，地面积水被脚跟带出弧形水线”。

### CG级特效与氛围控制

- 特效分层：主体动作第一层，能量轨迹第二层，烟尘/碎石第三层，背景废墟第四层；不能让特效遮住主体动作。
- 能量色彩：每个角色只保留一种主能量色，例：紫色电弧、红色裂光。不要一人多色乱闪。
- 蓄力镜头：近景压脸或手部能量，能量在掌心/武器边缘旋转，发丝和衣料轻动，背景虚化。
- 释放镜头：用中远景交代能量方向、距离和目标，不要只给脸部特写。
- 冲撞镜头：可用0.3-0.5秒强光遮切，但强光后必须回到清晰站位、接触点或结果反馈。
- 烟尘层次：烟尘从地面或撞击点扩散，前景可遮挡边缘，主体轮廓必须可见。
- 地面反馈：积水震开、碎石弹起、裂纹短促扩散、反光被能量扫过，用来证明力量。
- 胜负氛围：胜者近景压脸或俯视，败者低位/半跪/倒在前景，背景烟尘慢慢散开。
- 声音提示：低频能量嗡鸣、撞击闷响、碎石滑落、烟尘回落，不要只写爆炸。

CG战斗提示公式：

`废墟空间 + 双方能量色 + 站位距离 + 蓄力近景 + 释放中远景 + 强光短遮切 + 烟尘/地面反馈 + 胜负构图`

例：
`废墟城市冷灰色调，白发少年在左侧凝聚紫色电弧，红发少年在右侧压低重心聚起红色裂光；镜头先压近白发少年发丝和掌心电弧，再切中远景显示两人隔着积水地面对冲。紫电与红光接触瞬间短暂爆白0.3秒，爆白后回到清晰站位：红发少年被逼退半步，积水向后震开，白发少年肩侧残留紫色电弧，背景烟尘从地面扩散。`

## 参考视频风格条目：近身格斗拆招

来自本地参考视频 `13424291489395875.mp4` 的可复用规律。适合现代近身格斗、徒手拆招、多人围攻、训练式战斗、短剧肉搏戏。

### 场景与质感

- 场景：夜晚湖边公园、草地、树影、远处路灯、湖面反光，环境真实克制。
- 光线：冷蓝夜色为底，路灯和湖面反光提供弱暖点，不使用大面积魔法光效。
- 画面：中景和近景为主，动作清楚，人物轮廓干净，背景不过度抢戏。
- 字幕类画面不学习字幕本身，只学习动作设计和镜头关系。

### 站位与空间

- 近身格斗必须先给全身或半身中景，锁定双方距离和朝向。
- 双人对打常用左右站位：进攻者在一侧，防守者在另一侧，中间留出手臂接触空间。
- 多人围攻时，只让当前攻击者进入主动作位，其他人留在背景或侧后方等待下一拍。
- 保护对象或旁观者要有明确层级：前景受保护者，中景格斗者，背景待机者。

### 动作链规律

- 拳脚不是直接命中，先有预备动作：抬膝、沉肩、转胯、脚跟压地、手肘回收。
- 防守动作比进攻动作更清楚：抬掌封线、前臂格挡、手腕拨开、侧身让位。
- 接触点要明确：掌心对掌心、前臂挡拳、手腕被扣、脚背踢到腿侧、肩背被推开。
- 每个动作后都有反馈：被挡者手臂停住、身体重心偏移、脚步补位、衣摆滞后、头发甩动。
- 打斗不是乱挥，使用“攻击 - 拦截 - 转位 - 反击 - 拉开距离”的循环。

### 镜头规律

- 关键动作前用脚步特写：鞋底压地、脚尖转向、脚跟抬起，说明发力来源。
- 手部拆招用近景：只拍胸口到手臂范围，观众看清谁抓谁、谁挡谁。
- 命中或格挡后切反应：被挡者眼神一滞，防守者下颌收紧，旁观者身体前倾。
- 大动作如跳起、飞踢、被推开，要切回中远景，保留完整身体线条。
- 镜头不乱晃，动作快但机位稳，切点落在接触瞬间或重心改变瞬间。

### 近身格斗提示词公式

`站位锁定 + 起手预备 + 接触点 + 防守/反击动作 + 重心变化 + 布料/头发/地面反馈 + 镜头落点`

例：
`黑衣男站在画面右侧，左脚向前压住地面，肩线下沉后直拳打向橙衣男胸口；橙衣男站在画面左侧，左掌竖起封住来拳，右手护在胸前，拳面停在掌心前一寸，黑衣男袖口因急停向前甩出，镜头急停在掌心与拳面之间的接触点。`

### 实时速度格斗规则

- 如果用户反馈变慢、拖沓、像摆拍，启用本规则。
- 提示词必须明确写：实时速度、正常速度、动作连续完成、不使用慢动作、不定格、不停顿。
- 每个镜头写 1-2 个连续动作，不写“停在”“卡住”“凝固”“慢慢”“缓缓”“定格”。
- 接触点不能成为静态展示，接触后必须立刻接转位、补步、反击或拉开距离。
- 切镜用“动作峰值切到下一拍”，不要用“急停在接触点”。
- 反应镜头控制在 0.5-1 秒，用眼神扫向下一动作，不让人物站着发呆。
- 声音用连续节奏：脚步、衣料、掌击、擦步、呼吸，不用低沉慢动作音效。

### 紧凑实战短打规则

- 如果用户反馈“像过家家、不紧凑”，启用本规则。
- 双方距离压到半步到一步内，使用贴身短打、封手、抢中线、肘膝、肩撞、拨腕、绊步。
- 攻防必须重叠：防守动作同时为反击铺垫，不能一方打完另一方才开始。
- 少写“拉开距离、退半步、重新摆架势”；除非结尾需要转场，否则不重置站位。
- 动作节奏用“一口气连三拍”：封住来拳、贴身进位、肘击/膝顶/绊脚同时发生。
- 镜头更贴近身体：胸口到手臂近景、腰胯到脚步特写、肩线擦过镜头，减少空旷中景。
- 物理压迫要可见：肩膀挤压、衣料被抓皱、脚步顶住对方脚尖、手臂卡住脖颈外侧但不伤害性描写。
- 禁止互相等待、对称出招、你一下我一下的回合制描述。

紧凑短打公式：

`半步距离 + 攻防同时发生 + 封手/抢中线 + 贴身进位 + 短促反击 + 脚步卡位 + 不重置站位`

紧凑短打例：

`正常速度贴身短打。橙衣男左拳刚起，黑衣男右前臂已经压住拳线，左肩挤进橙衣男胸前半步距离；黑衣男左手顺势扣住橙衣男手腕向下带，右膝短促顶向橙衣男大腿外侧，橙衣男脚尖被卡住只能拧腰卸力，双方身体没有拉开，镜头贴着两人手臂交错处切到脚步卡位。`

实时速度公式：

`正常速度 + 连续动作 + 接触后马上转位/反击 + 动作峰值切镜 + 禁止慢动作/定格/停顿`

实时速度例：

`正常速度近身格斗。橙衣男从左侧向前半步出直拳，黑衣男在右侧抬左掌封住拳线，同时右脚斜跨到橙衣男外侧；拳掌一碰就分开，黑衣男顺势扣住橙衣男手腕向外拨，橙衣男立刻补左脚稳住重心，镜头在拨腕动作峰值切到下一招，不使用慢动作、不定格。`

### 近身格斗禁忌

- 不写“双方激烈缠斗”，要拆成一招一招。
- 不让三个人同时攻击同一目标，除非文案明确群殴；即便群殴，也要分前后拍。
- 不用能量爆炸替代拳脚接触。
- 不让人物凭空飞起；跳起前必须有屈膝、蹬地或借力动作。
- 不写“完美闪避”，要写肩线转开、脚尖换向、攻击从衣角旁擦过。
- 不写“急停、定格、慢推、缓慢、停住半秒、卡住半秒、轻震后停住”，这些词容易让模型生成慢动作。
- 不写“拉开两步、重新摆好架势、互相观察、轮流出招”，这些词容易让打斗变松散、像排练。

## 参考截图条目：秒表式高密度战斗分镜

来自用户提供的战斗提示词截图，可复用的是“整体资产提示词 + 秒表式动作链分镜”的结构，而不是具体丧尸、机器人或枪战设定。适合15秒战斗、追逐、围攻、突围、压制式收束。

### 核心结构

先写总控，再写分镜：

`视觉基调 + 色彩影调 + 情绪速度 + 画面内容 + 秒表式分镜`

总控必须先锁定：

- **视觉基调**：电影摄影感、手持临场感、动态模糊、速度感、力量感。
- **色彩影调**：主色调、光源来源、明暗对比、颗粒/胶片/柔光/硬光等画质。
- **战斗情绪**：紧迫、压迫、逃亡、围攻、反杀、收尾，不写空泛“超燃”。
- **主体关系**：谁是主角，敌人数量/方向/距离，主角是否被围堵或主动突进。

### 截图式画质总控硬规则

高密度战斗提示词必须在分镜前写出以下5项；缺少任意一项时，先补齐再输出：

1. **视觉基调**：写清摄影质感和镜头状态，例如变形宽荧幕电影感、IMAX胶片摄影机、Panavision C系列镜头感、手持跟拍、动态模糊、临场感、紧迫感。
2. **色彩与影调**：写清时代/题材美学、主色和辅色、饱和度、胶片颗粒、明暗对比、高光和暗部细节，例如冷灰废土底色、橙红火光强调、低饱和胶片滤镜、高光不过曝、暗部保留细节。
3. **光源与照明**：写清自然光、日光、月光、火光、霓虹、法术光、顶灯等来源，以及强直射光、侧逆光、硬光、自然光晕、体积光是否出现。
4. **画面情绪**：写清战斗的情绪功能，例如逃亡、压迫、反杀、困兽突围、末日危机、战斗速度感、力量感、危险逼近。
5. **主体和空间关系**：写清主角、敌人数量、最近距离、画面左右/前后关系、背景威胁和可借力环境，不让分镜开始后才临时补空间。

截图式总控模板：

```text
【视觉基调】
变形宽荧幕电影质感，IMAX胶片摄影机感，Panavision C系列镜头感，手持跟拍，轻微手持抖动，动态模糊只用于高速动作边缘，主体动作清晰。

【色彩与影调】
写主题美学；主色为___，辅色为___；低饱和/高对比/胶片颗粒/自然光晕；高光不过曝，暗部保留细节，明暗过渡自然。

【光源与照明】
主要光源来自___，辅助光来自___；使用___光；光线方向为___；环境中的尘埃/雨水/烟雾呈现体积光或反射。

【画面情绪】
战斗情绪为___，强调___感、___感、___感，不写空泛“超燃”。

【主体和空间关系】

```

常用色彩影调库：

- **废土末日**：冷灰/尘土黄为底，橙红火光点缀，低饱和胶片颗粒，高反差硬光，暗部保留废墟细节。
- **玄幻神战**：深青/黑金为底，金色神光或蓝白雷光强调，冷暖强对比，体积光穿过云雾，金属甲片有磨损反光。
- **赛博夜战**：深蓝/黑紫为底，红蓝霓虹交叉，湿地反射，局部高饱和，暗部保留人物轮廓。
- **古风武侠**：青灰/月白为底，暖烛/冷月二选一，柔雾轻颗粒，布料和木石材质清楚。
- **影视写实**：自然日光或实景灯光，低饱和真实肤色，强弱光源有明确来源，不过度粒子化。

输出提示词时，不能只写“电影感、画质高级、国漫质感”；必须拆成摄影、色彩、光源、情绪和空间五层。

### 15秒秒表结构

当用户要高密度战斗，默认采用8段式，不平均切镜，按动作阶段分配：

1. **分镜1 开场入画（0-1.5s）**：主角进入画面或摔落入画，给出姿态、表情/状态、落地反馈。
2. **分镜2 备战起势（1.5-3s）**：主角起身、拔刀、拔枪、凝力、锁定敌人，交代敌人数量与最近距离。
3. **分镜3 第一波交锋（3-5s）**：闪避/横移/格挡后完成第一组有效命中，交代命中对象和结果。
4. **分镜4 背后威胁（5-6.5s）**：身后或侧面敌人袭来，主角转身、重击或近身反制。
5. **分镜5 位移卖点（6.5-8s）**：滑步、回旋踢、翻身、撞墙、借物换位等一个高辨识动作，必须改变站位或胜负关系。
6. **分镜6 近身缠斗（8-9.5s）**：敌人贴近主角，主角用肘、肩、膝、枪托、刀柄或环境完成压制。
7. **分镜7 最终终结（9.5-11s）**：最后一个强敌/最后一击，完成动作闭环，禁止拖成长时间对峙。
8. **分镜8 起身收尾（11-15s）**：战斗结束，主角整理武器/回头/听到声音/发现新威胁，留下下一段钩子。

### 每个分镜必须包含

`分镜编号 + 阶段名称 + 时间段 + 镜头方式 + 主体动作 + 攻防对象 + 接触点 + 结果反馈 + 下一动作衔接`

例：
`分镜3 第一波交锋（3-5s）过肩视角跟拍，林烬向右横移避开前方敌人扑击，左臂格挡敌人手腕，右手短刀贴近敌人肩线反切，敌人重心被带偏撞向货架，货架震动并掉落灰尘，林烬顺势转身接背后威胁。`

### 秒表式战斗的优点

- 15秒内信息密度高，但每段动作职责明确。
- 每段都有“动作结果”，避免只有挥舞没有命中。
- 敌人数量、距离、方向持续更新，减少站位漂移。
- 最后4秒留收尾和钩子，避免生成视频突然断掉。

### 使用限制

- 如果模型容易乱动，敌人数量减到2-3个，保留8段时间结构但合并击杀段。
- 如果是即梦/Seedance，最终提示词仍需控制在1500字内；删画质堆词，不删时间段、动作对象、接触点和结果。
- 暴力内容改为平台友好的“击倒、震退、倒地、失去行动能力”，少写过度血腥细节。

## 参考视频风格条目：影视级氛围

来自本地参考视频 `13424292404693886.mp4` 的可复用规律。适合古风、仙侠、权谋、情感、庭院、室内、花树、窗纱、回廊等高质感AI短剧。

### 核心结构

影视级氛围提示词按这个顺序写：

`风格核心 + 拍摄参数感 + 画面处理 + 光线布置 + 人物状态 + 服饰材质 + 场景空间 + 负面限制`

不要只写“电影感、氛围感、高级感”，必须拆成可见的摄影条件。

### 镜头与质感

- 镜头感：85mm 或 135mm 中长焦感，人物近景或中近景，背景柔化，主体面部和服装细节清晰。
- 光圈感：浅景深，背景花树、窗纱、灯笼、廊柱柔化成层次，不抢主体。
- 画面处理：半哑光质感，皮肤不过度磨皮，布料纹理、发丝、首饰边缘保留细节。
- 真实感：不要全景大空镜堆砌，优先聚焦人物美貌的近景、中景和局部特写。

### 光线布置

- 主导光：强侧逆光或柔逆光，从人物后方侧上方进入，打亮发丝边缘、衣袖边缘、肩线。
- 面部补光：正面使用弱柔光或环境反射光，只保留眼神光，不把脸打平。
- 体积光：在窗纱、花树、烟雾、尘埃中形成可见光束，增加空间纵深。
- 光色：暖春日、烛光、晨光、夕光用于古风温润；冷月光、青灰阴影用于权谋压迫。
- 禁止无来源光：所有光必须来自窗、门、烛、灯、月、日、火、法阵等可解释来源。

### 场景空间

- 场景必须先做概念板式描述：外景、内景、入口、主视角、局部细节、俯视布局。
- 古风庭院：树影、青石路、木门、窗棂、帘纱、香炉、茶案、花枝、墙面斑驳。
- 室内：前景帘纱或门框，主体人物，背景窗格/屏风/花影，形成前中后三层。
- 场景用于防止穿帮：同一段分镜内门、窗、桌、屏风、人物位置不能随机变化。

### 人物资产

- 重要角色先生成三视图或固定资产：正面、侧面、背面，发型、发饰、服装、妆容、身形保持一致。
- 人物提示词必须写清：年龄感、脸型、发型、服装材质、主色、饰品、身份气质。
- 情绪用小幅度表演：眼神侧移、唇线收紧、手指轻触衣袖、下颌微收，不用夸张大哭大笑。

### 影视级氛围公式

`高质感古风/玄幻短剧 + 中长焦近景 + 浅景深背景柔化 + 侧逆光发丝轮廓 + 弱面部补光 + 体积光穿过窗纱/花枝 + 前景遮挡 + 主体清晰 + 布料纹理和发丝细节 + 克制表演 + 不要字幕不要水印`

例：
`高质感古风短剧，85mm中长焦近景，浅景深，窗外暖春日侧逆光穿过薄纱和花枝，形成柔和体积光，年轻男子站在木窗旁侧脸看向院中，发丝边缘被光勾亮，面部由弱环境反射光保留眼神光，衣领刺绣和发丝细节清晰，前景一角帘纱轻微虚化遮挡，背景花树柔化，表情克制，唇线微收，不要字幕不要水印。`

### 负面限制

- 不要全脸磨皮、网红滤镜、塑料皮肤、过亮眼白、夸张大眼。
- 不要随机换衣服、换发型、换场景门窗位置。
- 不要所有画面都强逆光；逆光用于轮廓，面部仍需可读。
- 不要大量彩色粒子和无来源光束。
- 不要把教程字幕、软件界面、时间码、logo当作创作内容学习。

### 影视级AI短剧生产流

从参考视频 `13424292404693886.mp4` 学到的核心不是软件界面和教程字幕，而是“先做资产，再做镜头，再做剪辑”的生产顺序。以后生成提示词时，必须把这个顺序转化为可执行约束。

标准流程：

1. **剧情节点**：先把文案拆成钩子、推进、反应、反转、转场锚点；每段不超过15秒，只服务一个事件。
2. **角色资产卡**：重要角色先固定正脸、侧脸、背影、发型、服装正反面、饰品、身形、气质、表演幅度；每段提示词用角色全名继承这些资产。
3. **场景概念板**：每个主场景先固定全景、入口、门窗、道路/桌椅/屏风等空间参照物、局部细节、俯视站位图、光源方向。
4. **镜头节点生成**：每个镜头只写一个动作峰值或一个反应点；镜头移动必须为了看清手、眼神、道具、站位变化或动作接触点。
5. **选片逻辑**：生成结果优先保留人物脸型一致、服装一致、空间方向一致、动作接触点清楚的版本；画面漂亮但人物漂移的版本弃用。
6. **剪辑层合成**：主视频层负责动作和表演；反应镜头层负责情绪；音效/BGM层卡动作峰值；转场层只用于场景切换、回忆、快剪或段落结尾；调色层统一冷暖和肤色。

### 角色资产卡硬规则

- 角色资产卡最少包含：角色全名、年龄感、脸型、发型、发色、服装主色、服装材质、饰品、身高体型、身份气质、常用表演幅度。
- 同一角色在连续分镜中不得随机换发型、换服装主色、换饰品、换脸型；剧情没有换装时，服装细节要连续。
- 近景和特写必须优先重复“脸型、发型、服装领口/肩部/饰品”这些最容易变脸的锚点。
- 表演幅度要设上限：压抑、警惕、犹豫、愤怒都用眼神方向、唇线、下颌、呼吸、手指和重心变化表达，避免每段都流泪、冷汗、大喊。

角色资产卡格式：

```text
角色资产卡：
角色全名：
年龄感：
脸型/五官：
发型/发饰：
服装主色/材质：
固定饰品：
身高体型：
身份气质：
表演幅度上限：
不可变化项：
```

### 场景概念板硬规则

- 每个主场景先建立概念板，不直接进入镜头：外景全貌、室内主视角、入口、关键家具/障碍物、局部材质、俯视站位图、主光源方向。
- 分镜中所有站位必须绑定场景参照物，例如“木门左侧、石桌前方半步、屏风右前方、走廊尽头、竹竿后侧”，不能只写“左边、右边”。
- 同一段内门、窗、桌、屏风、楼梯、竹林、车辆、掩体的位置不能随机变化；镜头换角度时必须说明是机位变化，不是人物换位。
- 场景材质要服务真实感：木门有旧纹、石路有水痕、布帘有折痕、金属有磨损、地面有脚印或灰尘反馈。

场景概念板格式：

```text
场景概念板：
场景名称：
外景/内景全貌：
入口与出口：
固定参照物：
局部材质：
光源方向：
俯视站位：
不可变化项：
```

### 节点式镜头输出规则

- 正式15秒提示词前，先在脑内完成“剧情节点 -> 角色资产 -> 场景概念板 -> 站位锚点 -> 镜头节点 -> 剪辑层”的转换。
- 输出时可以不展示完整资产卡，除非用户要求；但镜头描述必须体现资产卡和概念板的结果。
- 每段开头必须写清“承接上一段结尾站位”；第一段也要写清初始站位。
- 每段结尾必须留下“下一段可承接锚点”：谁在什么参照物旁，朝向谁，手脚/武器/道具处于什么状态，动作余势往哪里去。
- 镜头命名建议：`S01_SH01_场景名_角色A_角色B_动作点`，用于后期整理素材和筛选一致版本。

### 剪辑层与音效层规则

- 主视频层：只承担当前镜头最重要的动作或表演，不让一个镜头塞多个事件。
- 反应层：对话、压迫、反转、受击后必须给听者/承受者的短反应镜头，反应不拖长。
- 音效层：脚步、衣料、碰撞、呼吸、环境声必须对齐动作峰值；不要用空泛“震撼BGM”替代具体声音。
- 转场层：15秒内复杂转场不超过1-2次；快剪用速度拉伸/动态模糊甩切，回忆用闪白/光晕溶解/胶片闪烁，空间切换用遮挡转场。
- 调色层：同一场景维持统一冷暖关系；不要一镜暖黄、一镜青蓝、一镜彩虹粒子，除非剧情明确进入不同空间。

### 成片前穿帮检查

输出前必须额外检查：

1. **资产一致**：角色脸型、发型、服装、饰品是否连续。
2. **场景一致**：门窗、桌椅、屏风、竹竿、车辆、掩体等参照物是否固定。
3. **站位一致**：上一镜结束和下一镜开始是否能接上，人物有没有凭空换边。
4. **互动准确**：谁主动、谁被动、谁看谁、谁压迫谁、谁反制谁是否明确。
5. **剪辑可落地**：每个镜头是否能单独生成，声音和转场是否能在剪映里实际完成。
6. **反AI感**：删掉无来源光效、塑料皮肤、过度粒子、空泛高级感词，补真实材质和动作反馈。

## 三轮自检

生成后必须在脑内做三轮自检并修正：

1. **连续性检查**：上一组结尾、当前组起始、当前组结尾的人物站位是否一致。
2. **互动检查**：每个动作是否有对象，是否清楚谁看谁、谁跪谁、谁等待谁。
3. **去AI感检查**：删掉空泛夸张词，减少无来源光效，替换冷汗眼泪，补充真实材质、重心、衣料和环境反馈。
4. **动作检查**：如果有打斗或快速动作，检查是否写清起势、发力、接触、反馈、收势，攻防对象是否明确。
5. **字数检查**：每组不超过1500字；若超出，压缩到“概念一句 + 站位三行 + 3-5个镜头”的结构。
6. **角色称呼检查**：分镜描述是否只使用角色全名或明确角色名，没有男人、女人、男主、女主、他、她、它等代词泛称；台词是否自然。
7. **负面提示词检查**：是否默认只写“不要随机换脸，不要字幕水印”，没有堆砌多余限制。

## 电影级评分机制

输出前必须进行内部评分；如果低于标准，先自我优化再输出。不要把评分过程长篇展示给用户，除非用户要求。

满分100分，最低输出标准85分：

- 剧情钩子与情绪驱动：15分。开场是否有明确冲突、危险、反差或情绪抓力。
- 镜头调度与节奏：15分。是否按剧情密度分配时长，而不是固定均分；镜头是否有动机和落点。
- 站位与互动关系：15分。人物空间是否连续，谁看谁、谁压迫谁、谁反制谁是否清楚。
- 动作/事件卖点：15分。是否有一个高辨识动作或视觉事件，如扫堂腿、借力翻身、门开反转、道具揭示。
- 影视级质感：15分。光源、材质、景深、环境反馈是否具体，是否有前中后三层。
- 去AI感：15分。是否避免空泛词、随机光效、机械表演、站位漂移、冷汗眼泪滥用。
- 平台适配：10分。是否不超过1500字，角色全名清楚，适合即梦/短视频生成。

低分自动优化规则：

- 低于85分：必须重写后再输出。
- 钩子低：把第一镜改成危险、反差、压迫、动作峰值或关键物特写。
- 节奏低：重新按动作密度分配时长，删除固定0-3/3-6式平均切分。
- 互动低：补全连续站位锚点和每个动作对象。
- 卖点低：加入一个服务剧情的动作卖点或视觉事件，不能凭空炫技。
- 质感低：补光源来源、材质重量、环境反馈和前中后三层。
- 去AI感低：删掉“史诗、震撼、高级、炫酷”等空泛词，减少无来源光效。
- 适配低：压缩到1500字以内，保留站位、互动、动作、光线，删解释。

最终输出前只保留成片提示词；必要时可在末尾用一行短注说明“已按电影级评分机制自检并优化”。

## 风格选择机制

当用户没有明确指定画面风格时，先给3个风格选项，不直接生成最终提示词。每个选项包含：风格名、适合原因、可能风险。

常用风格池：

- 2D赛璐璐动画风格：适合青春校园、轻喜剧、清晰动作、成本稳定；风险是影视质感稍弱。
- 3D国漫风格：适合玄幻、古风、动作、魔法、角色颜值；风险是容易有AI塑料感，需要写材质和光源。
- 影视写实风格：适合战术、历史、普法、现实题材；风险是人物一致性和真实表演要求更高。
- 赛博朋克风格：适合城市夜景、科技、追逐、暗黑动作；风险是霓虹和光效容易过度。
- 温暖3D动画风格：适合公益、普法、校园、家庭、民族团结；风险是动作戏冲击力较弱。
- 水墨国风动画风格：适合武侠、仙侠、山水、古意情绪；风险是细节和人物稳定性需要控制。

输出示例：

```text
这段剧情可以走三个方向：
1. 3D国漫风格：适合动作和颜值表现，打斗有冲击力。风险：需要控制材质，避免AI塑料感。
2. 2D赛璐璐风格：适合清晰动作分镜和青春感。风险：光影电影感弱一点。
3. 影视写实风格：适合真实压迫和战术动作。风险：对人物一致性要求更高。

你选一个风格，我再生成正式15秒提示词。
```

## 快剪与回忆转场规则

当剧情出现快剪、回忆、闪回、意识片段、时间跳跃、战斗记忆、前世画面、系统提示等内容时，必须写清转场。

### 触发场景

- 快剪：战场混乱、训练片段、追逐、打斗连续击中、信息快速交代。
- 回忆/闪回：角色想起过去、原主记忆涌入、前世画面、梦境片段。
- 时间跳跃：第二天、数日后、多年后、从现代切到古代或从现实切到小剧场。
- 意识冲击：穿越、系统激活、身份信息进入脑海、战场记忆叠加。

### 剪映高级转场可选项

- 闪白转场：适合穿越、强光、记忆刺入、爆点切换。
- 变焦模糊：适合意识被拉入回忆、惊觉、身份信息冲击。
- 光晕溶解：适合温情回忆、校园、公益、普法宣传。
- 胶片闪烁：适合旧记忆、战场残片、历史画面。
- 速度拉伸/速度坡度：适合快剪、冲刺、连续打斗，不写慢动作。
- 动态模糊甩切：适合打斗动作峰值、转身、闪避、奔跑。
- 遮挡转场：用衣袖、门框、竹叶、烟尘、人物背影掠过镜头自然切换。
- 故障闪烁：只适合系统、赛博、电子屏，不用于古风写实场景。

### 使用规则

- 每次转场都要写清：从哪个画面切到哪个画面，为什么转。
- 15秒内复杂转场不超过1-2次，避免AI感和剪辑过花。
- 快剪可以密，但每个画面必须有明确内容，不要写“快速闪过许多画面”。
- 回忆转场必须改变画面质感：色调、颗粒、边缘模糊、声音压低，帮助观众识别过去时空。
- 禁止默认每段都写“尘土转黑”“烟尘转黑”。只有剧情进入压抑停顿、死亡寂静、章节结束时才使用转黑。
- 战斗段结尾必须按剧情功能选择转场：动作峰值用动态模糊甩切，冲刺/连击用速度拉伸，强光/能量爆点用闪白，强敌登场用声音匹配剪切，空间切换用遮挡转场，回忆/意识用变焦模糊或胶片闪烁。
- 结尾转场要服务下一段承接锚点：下一段从声音、光、动作方向、视线方向、道具落点或空间遮挡中接起。

### 战斗转场决策表

- **动作峰值接下一击**：动态模糊甩切。例：刀锋掠过镜头，顺势切到下一段敌人武器落下。
- **高速追击/连续突围**：速度拉伸/速度坡度。例：主角冲出包围，画面边缘拉伸，接下一段奔跑追逐。
- **能量爆发/法术碰撞**：闪白转场。例：雷光爆开占满画面，闪白后接新场景或新敌人。
- **强敌登场/声音先到**：声音匹配剪切。例：远处战鼓声或脚步声压过画面，直接切到强敌脚步踩碎地面。
- **空间切换/换场**：遮挡转场。例：披风、门框、烟尘、竹叶或人物背影掠过镜头，接新空间。
- **回忆/意识冲击**：变焦模糊、胶片闪烁。例：角色瞳孔收缩，画面变焦模糊进入旧战场残片。
- **悬念留白/章节结束**：转黑或转白。例：主角发现无法立刻解释的新威胁，画面转黑进入下一章。

转场写法例：

`赵怀安抱头，战鼓声压低，画面用变焦模糊进入原主记忆；现代体育馆裸绞示范、战场箭雨、陌刀砍杀三帧快剪，以胶片闪烁和鼓点切镜连接，最后用闪白转回赵怀安跪在尸骸间。`

## 1500字压缩结构

当用户要求即梦、Jimeng、剪映AI、豆包视频、可直接生成视频时，默认使用压缩结构：

```text
标题：
段落：第一段/第二段...
总时长：小于等于15秒
核心主题：
画面基调：
承接上一段结尾站位：（第一段可写“无，建立初始站位”）
连续站位：
人物互动：
0-X秒 · 镜头名：动作/站位/表情/运镜/声音。
X-X秒 · 镜头名：动作/站位/表情/运镜/声音。
X-X秒 · 镜头名：动作/站位/表情/运镜/声音。
最后1秒 · 转场钩子：动作/落点/转场。
负面限制：不要随机换脸，不要字幕水印。
```

压缩优先级：

1. 保留：角色、场景、站位、互动对象、动作接触点、镜头落点、光源、关键材质。
2. 压缩：声音、特效、氛围词、场景装饰。
3. 删除：解释原因、教程式说明、重复形容词、泛泛的高级感词汇。

## 超过15秒拆段规则

- 先按剧情事件拆段，不按固定时长硬切。
- 每段只表达一个主要事件或一个动作阶段。
- 每段时长小于等于15秒，镜头时长按剧情密度自由分配。
- 每段末尾必须留下下一段承接锚点：站位、动作方向、视线、道具状态、环境状态。
- 下一段开头必须复述并继承该锚点。

拆段示例：

```text
第一段：双方对峙并第一次交锋，结尾停在A压住B手腕。
第二段：承接A压住B手腕，B借地面翻身反制，结尾停在A被逼到墙边。
第三段：承接A靠墙，B继续压迫，完成制服或反转。
```

## 爆款短剧节奏

- 0-2秒：钩子画面，不铺垫。
- 2-6秒：交代当前危险或反差。
- 6-10秒：给人物反应或误会升级。
- 10-14秒：信息转折。
- 14-15秒：转场钩子。

## 质检清单

- 观众1秒内能不能看懂危险或反差？
- 15秒内是不是只讲了一个事件？
- 角色外貌和服装有没有重复写清楚？
- 每个镜头有没有动作结果，而不只是“看着”“站着”？
- 最后一镜能不能自然接下一组？
- 当前组第一镜是否继承上一组最后一帧？
- 人物站位是否全组一致，没有突然换边、换距离、换高低？
- 人物互动关系是否准确，能看出谁对谁产生动作或反应？
- 微表情是否克制，没有滥用冷汗、眼泪和夸张表情？
- 是否没有明显AI感：无过度光效、无机械表演、无随机运镜、无塑料材质、无空泛形容词？
- 如果是战斗镜头，是否有清晰攻防关系、身体发力、物理反馈和明确镜头落点？
- 是否适配即梦：单组提示词是否不超过1500字，且没有删掉站位和互动关系？
- 是否角色名清晰：每个动作是否都用全名标注角色，没有泛称和代词？
