# 第 3 项人物美型与画风适配设计

## 目标

优化“剧本资产提取”的人物结果，使主要角色具有精致、鲜明、可直接生图的角色设定质量，同时严格跟随用户选择的画风锚点。改动仅作用于人物提取，不改变场景、物品提取及人物三视图版式。

## 角色分级

- 男女主角、核心反派、重要配角使用主角级美型标准：五官精细、轮廓明确、气质突出，并为脸型、眉眼、发型、服装和身份细节设置独立记忆点，避免大众脸和角色撞脸。
- 普通成年配角保持协调、自然、有辨识度，不强制套用男女主角的精致模板，外貌与体态服从职业、阶层、生活状态和剧情功能。
- 老人、儿童、病弱者及特殊身份角色按年龄和剧情塑造，不套用年轻角色体态、华丽服饰或高开衩设计。
- 所有角色不得模仿现实明星、网红、艺人或博主，也不得使用同质化网红脸。

## 女性主角级标准

对于原文明确定义为成年女性且属于女主角、重要女性反派或核心女性配角的角色，在不违背原文年龄、身份、时代和剧情的前提下，参考用户提供的清雅佳人标准：

- 脸型精致、骨肉匀称、线条流畅，眉眼清晰灵动，眼神同时体现角色性格与剧情张力。
- 鼻唇、肤质、发型和配饰均写出可视化细节，形成独立辨识点。
- 体态优美、比例协调，可使用窄肩、收腰、修长腿部等美型语言，但不得覆盖原文明确设定。
- 服装按时代、身份和场景精细设计，写清剪裁、面料、颜色过渡、纹样、配饰和鞋履；华丽程度服从角色阶层，不让所有女性统一穿同类服装。
- 清雅、飒爽、温婉、冷艳、明艳等气质由剧情决定，不把所有女性固定成同一种佳人模板。

## 画风动态适配

人物审美要求只负责“角色是否精致、有辨识度”，不直接写死材质和渲染方式。材质、线条、色彩、光影及镜头语言必须由当前画风锚点决定：

- `3D国漫风格`：允许次世代高精度建模、精细角色雕刻、次表面散射、发丝细节、PBR织物和国风材质等语言。
- 其他 3D 风格：使用该风格自己的几何比例、表面材质和渲染语言，不照搬 3D 国漫古风材质。
- 影视写实类：使用真人皮肤、真实布料、摄影和镜头光影语言，不混入 3D 建模或动画材质。
- 水墨、赛璐璐、手绘、漫画、像素等非 3D 风格：使用对应的笔触、线条、色块、纸张或动画语言，不出现 PBR、次表面散射、真人摄影和影视写实等冲突词。
- 输出的“整体风格”首句明确写出当前画风锚点，禁止固定追加与所选画风冲突的通用风格词。

## 图片结构

继续严格使用当前人物三视图参考图结构：纯白背景，上方三分之一为正面脸部近景头像，下方三分之二为颈部以下的正面、侧面、背面三个等比例竖向面板。顶部头像与下方身体视图必须保持同一角色、同一服装、同一画风、同一光影和统一人体比例；不出现字幕、水印、Logo、编号或面板标题。

## 生图链路

人物资产提取阶段生成包含角色等级、美型细节和动态画风语言的完整描述。点击生图时，生图提示词继续携带该资产完整描述，并额外锁定当前画风锚点和三视图结构，避免二次提示覆盖人物细节或混入其他风格。

## 验证

- 测试 3D 国漫女性主角提示词包含主角级精致度、原创辨识度和与该风格匹配的建模材质语言。
- 测试普通配角、老人和未成年角色不会被强制套用女性主角体态与服装标准。
- 测试水墨、赛璐璐和影视写实画风只包含各自允许的风格语言。
- 测试人物生图提示词保留角色精细描述、所选画风锚点和现有三视图结构。
- 确认场景和物品提示词不受影响。

## 不在范围内

- 不修改生图模型、分辨率、比例、并发或重试逻辑。
- 不改变人物三视图布局。
- 不改变第 3 项以外的功能。
