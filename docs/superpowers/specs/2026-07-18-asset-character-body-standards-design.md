# 第 3 项人物体态标准设计

## 目标

让“3 剧本资产提取”的每个人物结果都明确输出并在生图时执行独立的 `体态标准：` 字段，避免体态要求埋在长篇画风描述中而被模型忽略。

本次只修改人物资产提取及人物生图提示词，不改变场景、物品或其他功能。

## 输出结构

人物结果字段顺序调整为：

1. `角色等级：`
2. `人物外貌：`
3. `体态标准：`
4. `整体风格：`
5. `人物的身份：`
6. `图片的结构：`

`体态标准` 必须写成该角色可直接生图的完整描述，不能只写“按统一标准”“身材好”或省略字段。

## 体态路由

### 普通成年女性

除未成年人、老人、病弱、残障或其他明确特殊身份外，强制使用：

- 饱满 S 曲线
- 窄肩蜂腰
- 圆润胯部
- 前凸后翘
- 修长笔直大长腿
- 站姿自然挺拔，正面、侧面、背面比例一致

服装继续保持精美华丽的产品目标，可按角色差异使用收腰、高开衩、薄纱、刺绣、鎏金等设计；时代、身份、场景和剧情决定具体款式、颜色、露肤度与材质组合。

### 古风成年男性

当原文时代或角色造型属于古风、仙侠、玄幻、权谋、武侠等方向时，使用：

- 标准宽肩窄腰
- 挺拔直角肩线
- 躯干修长匀称
- 线条利落干净
- 肌肉轮廓流畅、不臃肿
- 身姿端正俊朗
- 笔直修长双腿

古风服装采用挺括修身版型，具体劲装、长袍、战袍或锦衣由身份和剧情决定。

### 现代成年男性

当原文时代或角色造型属于都市、现代、职场、校园成年角色等方向时，使用：

- 宽肩阔背
- 紧致窄腰
- 躯干挺拔匀称
- 体态干净利落
- 肌肉线条紧致自然、不夸张不臃肿
- 比例协调
- 修长笔直大长腿

现代服装采用修身立体、版型利落的设计，具体通勤、街头、工装或轻奢造型由身份和剧情决定。

### 例外角色

未成年人、老人、病弱、残障及其他明确特殊身份必须按原文年龄、身体状况和剧情塑造，不套用普通成年男女体态。原文明确的男性矮壮、魁梧、瘦小或年长体态优先于统一男性标准。

## 风格适配

体态标准描述人物比例和轮廓，不决定渲染材质。所选画风仍决定皮肤、建模、线条、材质、色彩和光影：

- `3D国漫风格` 使用国漫次世代人体与材质语言。
- `3D仿真精致角色` 使用高精度仿真人体与皮肤语言。
- `现代甜酷3D乙游` 使用乙游角色比例与时尚材质语言。
- 皮克斯和低多边形将同一体态目标转换成各自的动画比例或几何轮廓，不混入仿真材质词。
- 非 3D 画风仍保留体态字段，但用当前画风的线条和造型语言表达。

## 生图补全

人物生图提示词必须把 `体态标准` 放在画风规则和三视图结构之前，提高执行优先级。

如果旧结果或模型输出漏掉 `体态标准：`：

1. 软件读取人物描述中的性别、年龄、身份、时代和服装线索。
2. 明确普通成年女性时补入女性标准。
3. 明确成年男性时，根据古风或现代线索补入对应男性标准。
4. 明确未成年人、老人、病弱、残障或特殊身份时，不自动套统一体态，只补入“严格按原文年龄与身体状况塑造”。
5. 无法可靠判断性别或时代时，使用中性兜底：“体态严格服从原文年龄、身份和剧情，正侧背比例一致”，不得猜测成年女性或男性标准。

已有 `体态标准：` 时必须原样保留，不重复追加。

## 三视图一致性

- 顶部头像与下方三视图必须是同一角色。
- 下方正面、侧面、背面必须执行同一体态标准和人体比例。
- 不得出现正面纤细、侧面肥胖或背面肩胯比例变化。
- 双手自然下垂，双脚完整可见，鞋履符合时代与身份。
- 不改变现有顶部头像加下方三个无头身体视图的排版结构。

## 测试范围

- 人物提取提示词强制要求独立 `体态标准：` 字段。
- 普通成年女性、古风成年男性、现代成年男性分别包含完整标准。
- 未成年人、老人、病弱、残障和特殊身份不会被套用普通成年标准。
- 人物生图保留已有体态字段，并在漏写时按角色线索补全。
- 无法判断时使用中性兜底，不错误猜测性别。
- 体态字段位于画风和三视图说明之前。
- 场景和物品提示词不出现体态标准。
- 全量测试和生产构建通过。
